# ASAIS Group Website

Separate Website für ASAIS Group - Ein Kleingewerbe.

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

Die Website läuft auf Port 5174.

## Build

```bash
npm run build
```

## Technologien

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
